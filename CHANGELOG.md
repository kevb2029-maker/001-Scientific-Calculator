# Changelog

## [1.1.0] - 2026-06-15
### Changed
- Updated color palette to modern dark theme (deep slate, indigo accents, violet highlights)

## [1.0.0] - 2026-06-15
### Added
- Initial release
- Basic arithmetic operators with live expression display
- Scientific functions: sin, cos, tan, log, ln, sqrt, abs, factorial
- Inverse trig via INV toggle
- DEG / RAD mode toggle
- Memory operations: MS, MR, M+, M−, MC
- π constant
- Parentheses support
- Keyboard input support
- Responsive layout for mobile
